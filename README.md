# Cat Pinterest

Приложение для просмотра котиков с возможностью добавлять их в избранное.

## Функциональность

- просмотр списка котиков  
- бесконечная прокрутка  
- добавление и удаление из избранного  
- сохранение избранных котиков на клиенте  
- страница 404  
- placeholder, если изображение не загрузилось  

## Стек

- React  
- TypeScript  
- React Router  
- SCSS  
- Vite  

## Запуск проекта

1. Склонировать репозиторий:

    git clone https://github.com/Reu04/Cat-pinterest.git  
    cd Cat-pinterest  

2. Установить зависимости:

    npm install  

3. Запустить проект:

    npm run dev  

4. Открыть в браузере:

    http://localhost:5173/Cat-pinterest/  

## Сборка проекта

    npm run build
